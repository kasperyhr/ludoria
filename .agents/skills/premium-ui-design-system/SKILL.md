# 高级 UI 设计系统

## 什么时候触发

当任务涉及视觉设计、组件、主题、动效或页面 polish 时触发。

## 开发规则

采用 premium board game lounge、warm dark theme、soft amber accent、glass panel、subtle texture、calm animation。

## 禁止事项

不要做单一色块页面、过度圆角、重复卡片堆砌或不适合游戏大厅的营销空壳。

## Checklist

- 颜色统一
- 字体统一
- 间距统一
- 圆角和阴影克制
- 动效平静

## 项目示例

示例：房间页桌面端中央棋盘，侧边信息面板使用克制玻璃效果和琥珀焦点色。
