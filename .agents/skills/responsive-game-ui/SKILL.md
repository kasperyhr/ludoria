# 响应式游戏 UI

## 什么时候触发

当任务涉及移动端、桌面端、棋盘、手牌、聊天、日志或 puzzle board 布局时触发。

## 开发规则

移动端优先触摸可用性，桌面端优先信息密度和空间组织。

## 禁止事项

不要让文字重叠、控件跳动、棋盘溢出或把桌面布局硬塞到手机。

## Checklist

- 手机是否有底部操作栏
- 聊天是否可折叠
- 手牌是否可滑动
- 桌面是否有中央棋盘和侧边面板

## 项目示例

示例：手机端把聊天收进抽屉，底部固定主要行动按钮。
